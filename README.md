# TEMPORARY USE
1) Open Echo node with Bambu studio slicer and swap printer to P1S for SPARC 
2) chose which variant you'd like to print by moving the other variants to a new buuild plate.
    no parentheses = plain = recommended
3) select desired filament colors and then print
4) assembley and other components coming later
